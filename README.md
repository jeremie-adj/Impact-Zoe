# Impact Zoé

Site vitrine officiel d'Impact Zoé - Centre de couture, coiffure et formation à Tahoua, Niger.

## À propos
Impact Zoé accompagne les femmes de Tahoua avec :
- **Couture sur mesure** : robes, boubous, tenues traditionnelles et modernes
- **Coiffure professionnelle** : tresses, soins, coiffures d'événement
- **Formation** : apprentissage pratique en couture et coiffure
